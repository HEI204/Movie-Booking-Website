CW1

General Design

  • I use <header> tag for showing and grouping the header information for the page such as logo, company name and navigation bar.
  • For the navigation bar, i used the <nav> for the part of navigation bar and inside the nav use <a> for hyperlink
  • And also use multiple <div> tag to seperate and apply style to different part later more easily
  • Also, I use the footer tag to store the copyright information as normally this kind of information is stored in the footer part

Home page

  • The first div used to group the showing video and its information so that I can apply style to this part more easily
  • The second div used to store the multiple div about the information for two type of movie (Now Showing, Up Coming) so that I can apply different style easily

Now Showing page

  • I use multiple to <div> tag to seperate different main part in the page such the map, selecion of location, form for buy ticket
  • Also use <form> to allow user to select the data and time for their prefered movie and then click the button for submission

Ticket page

  • I use table to simulate the seating plan in cinema because it is easy to present the seating plan
  • Also create a form for later design to allow the user to submit their booked seat

Print page

  • I use different <div> tag to seperate the tickets and use <p> tag inside each <div> for showing the individual ticket information
